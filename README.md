Objective-C ORM
===============

Just another ORM.

## Can persist

* NSString
* NSNumber
* Bean
* NSArray of Bean

Bean = Entity

## Constraints

* A week
* No library
* Threads
* Can persist and load simple objects (String, Int...)
* Can persist and load complex objects (OneToMany, ManyToOne...)
* Easy to change adapter
